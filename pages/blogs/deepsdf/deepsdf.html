Deep SDF

0. Abstract

learned continuous signed distance function representation of a class of shapes.

1. Introduction

  • Deep convolutional networks are not scalable to 3D data with arbitrary topology.
  • Representation learns a generative model for a continuous SDF as opposed to discretizing the volume into a grid.
  • It is a learned shape conditioned classifier for which the surface is the decision boundary of the classifier.
  • memory footprint reduced by more than 50%

2. Related work

2.1 Representations for 3D shape

  • Point Based: Pointnet uses maxpool to extract global features, cannot describe topology and watertight surfaces
  • Mesh-based: predefined template meshes for human body reconstruction, describe 3D surfaces by using sphere parameterization (limited to topogological sphere), use convolution and pooling for meshes.
  • Voxel-based: Dense occupancy grids grow cubically, dense SDF are also memory intensive, octree based still have fixed resolution, no generalization over class of shapes.

2.2 Representation Learning Techniques

  • GAN: Generates high quality reconstructions, but training is unstable.
  • Auto-encoders: Variational auto-encoders use a gaussian perturbed bottleneck layer to encourage smooth latent spaces. Regularization over latent space allows exploring embedding space with gradient descent or random sampling.
  • Optimizing Latent Vectors: Train only decoder networks. search over space of latent vectors with fixed decoder parameters for inference.

2.3 Shape Completion

  • classical methods use radial basis functions to approximate dense surfaces (Poisson surface reconstruction)
  • Use data-driven approaches to fill in occupancy voxels, discrete SDF voxels, depth maps, etc.

7. Conclusion and Future work

Explicit optimization over latent vector during inference is undesirable. Possibly change ADAM optimizer with Gauss newton for faster convergence. DeepSDF also requires a canonical pose, and optimization over shape is possible only if the partial pointcloud is already aligned to this canonical pose.
