Superglue: Learning Feature Matching with Graph Neural Nets

1. Introduction

Learning feature matching can be treated as a linear assignment problem, and can be further relaxed into an optimal transport problem

Optimal Transport Problem: Generalization of the above linear assignment problem but sets are probability density functions, and the assignment is continuous. This makes the problem differentiable.

The cost function for optimal transport to be minimized is predicted by the Graph Neural net (GNN). The GNN uses self-attention (within image) and cross-attention (inter-image) to use both spatial and visual features of the keypoints. The Method is trained end-to-end from image pairs.

2. Superglue Architecture

2D Keypoints are generally projections of salient 3D points, like corners or blobs. This results in two constraints:

  1. A keypoint can have at most a single correspondence in the other image
  1. Some keypoints will not be matched owing to occlusions or detector failure

Formulation: Given two images A and B, each with sets of keypoints ppp﻿, and associated visual descriptor ddd﻿, call (p,d)(p, d)(p,d)﻿ as local features. Keypoints consist of (x,y,c)(x, y, c)(x,y,c)﻿ for position and detection confidence respectively. Visual descriptors di∈RDd_i \in \mathbb{R}^Ddi​∈RD﻿ can be obtained from CNNs or traditional descriptors

Partial Assignment: Owing to constraints 1 and 2. We have a partial assignment problem. Define a partial soft assignment matrix P∈[0,1]M×N\mathbf{P} \in [0, 1]^{M \times N}P∈[0,1]M×N﻿ such that P1N≤1M\mathbf{P1}_N \leq \mathbf{1}_MP1N​≤1M​﻿ and P⊤1M≤1M\mathbf{P}^\top\mathbf{1}_M \leq \mathbf{1}_MP⊤1M​≤1M​﻿.

2.2. Attention-based Graph Neural Network

Motivation: Humans look at contextual (relationship with other closeby keypoints) cues in images other than position and appearance to distinguish an image keypoint. Given an ambiguous keypoint, we look back and forth between the given image pair, to look for discerning features to disambiguate it. This suggests an iterative attention mechanism.

Multiplex Graph: A graph with two types of un-directed edges, usually described as multi-layer graph. One type of edge (self-edge) models inter-dependency within a single graph, and other type of edge (cross-edge) models dependency between sets of graphs.

Multi-head attention: Generally, multiple projection matrices are learned and multiple key value pairs are generated, which are concatenated and passed through a linear layer. This is similar to attention with different contexts in a given scene. For example: a feature keypoint, may have different attention w.r.t rotational invariance, and photometric invariance.

The final matching descriptors output from the GNN are linear projections of the last layer.

2.3. Training and Loss

Since both the graph neural network and optimal transport layer are differentiable, backpropagation is possible from the matches to the visual descriptors. Superglue is trained in a supervised manner, with ground truth matches M=(i,j)⊂A×B\mathcal{M} = {(i, j)} \subset \mathcal{A} \times \mathcal{B}M=(i,j)⊂A×B﻿. The matches are estimated via ground truth relative poses. Also a set of occluded keypoints are labelled manually as I⊆A\mathcal{I} \subseteq \mathcal{A}I⊆A﻿ and J⊆B\mathcal{J} \subseteq \mathcal{B}J⊆B﻿. 4. Conclusion and Key takeaways

  1. Increasing the receptive field of key-points via Graph neural network message passing, or Dynamic Graph CNN gives global context and allows to disambiguate locally similar key-points.
  1. Positional encoding of the features is important and has recently become very popular. NeRF, Fourier Features, BERT
  1. The general method of optimal transport can be applied to compute a measure between two probability distributions, be it discrete-discrete (Super-glue), discrete-continuous (density estimation problems), or continuous-continuous.
